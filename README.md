# BlogAPI

BlogAPI é uma API REST desenvolvida em **ASP.NET Core 6**, criada para gerenciar posts, categorias, autores e comentários de um blog.  
O sistema oferece funcionalidades de autenticação via **JWT**, permitindo que usuários se registrem, façam login e interajam com os conteúdos de forma segura.

O projeto foi pensado para ser **flexível e escalável**, facilitando futuras integrações e adições de recursos, como administração de usuários e gerenciamento de métricas.

## Principais funcionalidades

- Gerenciamento de posts, categorias e autores  
- Cadastro e autenticação de usuários com JWT  
- Criação de um usuário administrador inicial  
- Estrutura organizada, seguindo boas práticas de desenvolvimento  

O código é estruturado de forma clara, com atenção à manutenção e à segurança, utilizando ferramentas modernas como **Entity Framework Core**, **SQL Server** e **BCrypt** para hash de senhas.
